Instrument Basement

Instrument App

Instrument basement was my first React App and my first time implementing JSON web tokens for Authentication. Instrument Basement was an app designed for people to buy and sell instruments. Features included the ability for a user to sell an instrument and add instruments to a shopping cart for purchase. Very simple small app but a fun start and intro into React.
